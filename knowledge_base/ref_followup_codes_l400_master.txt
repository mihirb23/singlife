1. CONFIRMED CRITICAL CODES (frequently referenced in SOPs):
   - CSL : CareShield Plan Status — outstanding triggers Step 7B failure
   - C09 : Compliance GNS Rescreening — outstanding triggers TriggerGNS decision
   - F45 : Premium mismatch — common follow-up
   - GNS : Global Naming Screening — compliance hit
   - AT3 : DVM Agent Tier T3 — agent tier flag

2. STATUS CODES:
   - O = Outstanding (not yet resolved)
   - R = Resolved
   - Step 7B passes only when ALL follow-ups have status R

3. CATEGORY GROUPS:
   - SOL UWQns : Statement of Life - UW Questionnaires (any condition-specific UW form)
   - APS : Attending Physician Statement (specific to a medical condition K-code)
   - Pol Serv : Policy Servicing (post-issue changes)
   - MPP : Multi-Payor / High Net Worth requirements
   - DSP MLTC : Direct Sales Pre-MLTC checks

   - Look up the code here for its official L400 description
   - For frequency-of-use, see ref_followup_codes.txt (observed in MPCI dataset)
   - For CSL/C09/F45/AT3/GNS specifically, see ref_followup_codes_official.txt (with full SOP context)
