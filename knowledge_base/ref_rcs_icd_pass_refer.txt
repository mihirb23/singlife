# RCS ICD Code Pass/Refer Mapping
# Referenced by: Step 8H in SOP-NBIG-STP-001

## Purpose
Definitive pass/refer classification for ICD codes used in RCS (Risk Claims System)
checks. When Step 8G triggers RCS (only Claim Ind = Y, all other indicators = N),
the ICD codes from the shield claim are evaluated against this mapping.

## How RCS Check Works
1. Step 8G determines that only Claim Ind = Y and all others (Sub-std, Decline, Postpone) = N
2. System retrieves the ICD codes from the client's prior shield claims
3. Each ICD code is checked against the pass/refer mapping below
4. If ALL codes = Pass → Decision: Standard (STP)
5. If ANY code = Refer → Decision: Refer to UW

## ICD Code Classification

### PASS — Standard (STP eligible, no UW review needed)

| M95 | Other acquired musculoskeletal deformities | Structural, non-progressive |
| D36 | Benign neoplasm (other sites) | Benign = non-cancerous |
| D24 | Benign neoplasm of breast | Benign = non-cancerous |
| H58 | Other eye disorders | Low systemic risk |
| H93 | Other ear disorders | Low systemic risk |
| L30 | Other dermatitis | Skin condition, low risk |
| N73 | Female pelvic inflammatory disease | Treatable, low LTC risk |
| R07 | Pain in throat and chest | Symptom only (not diagnosis) |
| Z83 | Family history of disorders | History only, not active condition |
| Z04 | Examination and observation | Screening, no active condition |
| Z68 | Body mass index (BMI) | Indicator only, not a disease |

| F09 | Unspecified mental disorder | Mental health — affects ADL capacity |
| I51 | Heart disease complications | Cardiovascular — high mortality/morbidity |
| I10 | Essential hypertension | Cardiovascular risk factor — needs UW assessment |
| I77 | Other arterial disorders | Vascular disease — systemic risk |

## Multi-Code Cases
When a client has MULTIPLE ICD codes:
- If ANY single code = Refer → entire case = Refer to UW
- Pass codes do not override Refer codes
- The combination of codes may increase risk (e.g., E14 + E78 = diabetes + cholesterol)

## Important Notes
- This mapping is derived from the NB.MASTER dataset (807,851 records) and the
  Processing Steps matrix provided by Singlife Ops (Sharon/Terence)
- AI uses this mapping result only — AI does NOT make medical judgements
- If an ICD code is encountered that is NOT in this mapping, flag as Knowledge Gap
  and default to REFER TO UW (conservative approach)
- For Sprint 2 prototype: this mapping covers all 26 ICD codes observed in actual data

## Channel Applicability for Step 8H
