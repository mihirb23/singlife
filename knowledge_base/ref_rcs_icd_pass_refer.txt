# RCS ICD Code Mapping — ALL Codes = Refer to UW
# Referenced by: Step 8H in SOP-NBIG-STP-001
# Source: RCS_8H_ICD_Groups (Singlife official classification)

## Purpose
Definitive classification for ICD codes used in RCS (Risk Claims System) checks.
When Step 8G triggers RCS (only Claim Ind = Y, all other indicators = N),
the ICD codes from the shield claim are evaluated against this mapping.

## CRITICAL RULE
ALL ICD codes in the RCS system = Refer to UW.
There are NO pass codes. Any ICD code presence triggers underwriter review.

## How RCS Check Works
1. Step 8G determines that only Claim Ind = Y and all others (Sub-std, Decline, Postpone) = N
2. System retrieves the ICD codes from the client's prior shield claims
3. ALL ICD codes = Refer to UW (no exceptions)
4. Decision: Refer to UW

## Multi-Code Cases
When a client has MULTIPLE ICD codes, ALL codes still = Refer to UW.
Multiple codes may indicate higher complexity for the underwriter.

## Unknown ICD Codes
If an ICD code is encountered that is NOT in this mapping:
- Default to Refer to UW (conservative approach)
- Flag as Knowledge Gap for the UW team to add to the mapping
