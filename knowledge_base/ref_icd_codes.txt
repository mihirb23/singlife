====================================================================
RCS CHECK EVALUATION — ALL CODES = REFER TO UW
====================================================================

When Claim Indicator = Y in a case, the RCS (Risk Claims System) check
evaluates the ICD codes. Per the official Singlife RCS_8H_ICD_Groups
classification:

** ALL ICD CODES = REFER TO UW **

There are NO pass codes. Any ICD code presence in a shield claim
triggers mandatory underwriter review. This applies to all medical,
lifestyle, and occupational codes without exception.

For the complete ICD code mapping with descriptions and categories,
see: ref_rcs_icd_pass_refer.txt

Source: RCS_8H_ICD_Groups (Singlife official classification, confirmed
April 2026). This supersedes any previous pass/refer classification
that was inferred from data patterns.
